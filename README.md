#### [name for .exe] - the name of the compiled one .exe file (пример: main)
```commandline
gcc main.c -o [name for .exe]
```
#### Automatic file name a.exe
```commandline

```

### Running compiled code
#### [name for .exe] - the name of the compiled one .exe file (пример: main)
```commandline
./[name for .exe]
```
